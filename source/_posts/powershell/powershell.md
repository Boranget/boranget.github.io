# 批量重命名文件

`-replace`可使用正则表达式

注意必须使用单引号

```powershell
Get-ChildItem -Filter "*.txt" -File | ForEach-Object{
    $newname = $_.Name -replace '-(.*)-', '$1'
    Rename-Item $_.FullName $newname
}
```
