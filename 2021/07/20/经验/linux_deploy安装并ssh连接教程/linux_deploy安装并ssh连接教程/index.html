еҗҢж—¶жң¬ж–ҮдёӯдёҖдәӣеӣҫзүҮжқҘиҮӘдәҺиҝҷдәӣж•ҷзЁӢ

еҮҶеӨҮе·ҘдҪң

жңәеһӢйҖүжӢ©

иҝҷйҮҢе»әи®®йҮҮз”Ёй…ҚзҪ®иҫғй«ҳпјҢе®үеҚ“зүҲжң¬иҫғж–°дё”еҶ…еӯҳиҫғеӨ§зҡ„жүӢжңәпјҢжңҖеҘҪж”ҜжҢҒеҶ…еӯҳеҚЎжү©еұ•зҡ„жүӢжңәгҖӮжң¬жЎҲдҫӢдҪҝз”Ёзҡ„жңәеһӢдёәе°Ҹзұі3з§»еҠЁзүҲпјҢиҜҘжңәеһӢе®үеҚ“зүҲжң¬иҫғж—§пјҢжүҖд»ҘеңЁжүҫеҗҲйҖӮзҡ„Linux deployзүҲжң¬ж—¶иҠұдәҶдёҚе°‘ж—¶й—ҙгҖӮдё”иҜҘжңәеһӢдёҚж”ҜжҢҒеҶ…еӯҳеҚЎжү©еұ•гҖӮиҜҘжңәеһӢе®үеҚ“зі»з»ҹдёә4.4.4пјҢmiui8ејҖеҸ‘зүҲгҖӮ

жүӢжңәroot

е…¶д»–жңәеһӢзҡ„жүӢжңәиҜ·иҮӘиЎҢжҗңзҙўrootж–№жі•пјҢе°ҸзұіжүӢжңәйңҖиҰҒзі»з»ҹдёәејҖеҸ‘зүҲпјҢеҰӮжһңдҪ зҡ„жүӢжңәдёәзЁіе®ҡзүҲиҜ·жүӢеҠЁеҲ·жҲҗеҗҲйҖӮзҡ„ејҖеҸ‘зүҲгҖӮеҲҮи®°еҲ·жҲҗејҖеҸ‘зүҲд№ӢеҗҺдёҚиҰҒеҶҚзӮ№еҮ»еҚҮзә§зі»з»ҹдәҶпјҢдёҚз„¶еҫҲеӨ§еҸҜиғҪдјҡеҚҮжҲҗзЁіе®ҡзүҲгҖӮе°ҸзұіжүӢжңәејҖеҸ‘зүҲеҸҜд»ҘзӣҙжҺҘиҺ·еҫ—rootжқғйҷҗгҖӮд№ӢеҗҺйңҖиҰҒз»ҷbusyboxе’Ңlinux deploy rootжқғйҷҗгҖӮ

busybox дёӢиҪҪ

busyboxдёҖиҲ¬жүӢжңәзҡ„еә”з”Ёе•Ҷеә—е°ұжңүпјҢзӣҙжҺҘдёӢиҪҪе®үиЈ…еҚіеҸҜ

Linux deploy дёӢиҪҪ

йҰ–е…ҲиҙҙдёҠlinuxdeployзҡ„дёӢиҪҪең°еқҖhttps://github.com/meefik/linuxdeploy/releases
жіЁж„ҸиҰҒйҖүжӢ©йҖӮеҗҲиҮӘе·ұжүӢжңәзҡ„зүҲжң¬пјҢжң¬жЎҲдҫӢдҪҝз”ЁдәҶ2.2.2зүҲжң¬

й…ҚзҪ®

е®үиЈ…busybox

busyboxе®үиЈ…appе®ҢжҲҗеҗҺпјҢжү“ејҖпјҢзӮ№еҮ»е·ҰдёҠи§’зҡ„и®ҫзҪ®жҢүй’®пјҢжүҫеҲ°и®ҫзҪ®йҖүйЎ№гҖӮи®ҫзҪ®иҜӯиЁҖдёәдёӯж–Үз®ҖдҪ“гҖӮе…¶д»–й…ҚзҪ®й»ҳи®Өе°ұеҘҪпјҢиҝ”еӣһдё»з•ҢйқўзӮ№еҮ»е®үиЈ…гҖӮпјҲе®үиЈ…и·Ҝеҫ„дҪҝз”Ёй»ҳи®Өй…ҚзҪ®/system/xbinпјү


й…ҚзҪ®Linux deploy

linux deployи®ҫзҪ®

linux deploy е®үиЈ…appе®ҢжҲҗеҗҺпјҢжү“ејҖеә”з”ЁпјҢзӮ№еҮ»е·ҰдёҠи§’дёүдёӘжЁӘжқ еҗҺзӮ№еҮ»и®ҫзҪ®пјҢйҖүдёӯй”Ғе®ҡwifiгҖҒдҝқжҢҒCPUе”ӨйҶ’гҖҒеұҸе№•еёёдә®пјҲжңүзҡ„жүӢжңәеңЁжҒҜеұҸзҠ¶жҖҒдёӢCPUдјҡйҷҚйў‘иҝҗиЎҢпјүпјҢPATHеҸҳйҮҸдёҺBusyBoxдёӯй…ҚзҪ®дёҖиҮҙ /system/xbinпјҢи®ҫзҪ®е®ҢжҲҗзӮ№еҮ»жӣҙж–°зҺҜеўғгҖӮе…¶д»–и®ҫзҪ®зңӢдёӘдәәжғ…еҶөй…ҚзҪ®

й…ҚзҪ®ж–Үд»¶и®ҫзҪ®

е·ҰдёҠи§’дёүйҒ“жқ йҖүжӢ©й…ҚзҪ®ж–Үд»¶пјҢзӮ№еҮ»ж·»еҠ пјҢиө·дёӘеҗҚеӯ—еҰӮlinuxпјҢйҖүдёӯдҪ ж–°е»әзҡ„иҝҷдёӘй…ҚзҪ®ж–Үд»¶пјҢиҝ”еӣһгҖӮ
зӮ№еҮ»еҸідёӢи§’и®ҫзҪ®еӣҫж ҮпјҢи®ҫзҪ®еҰӮдёӢпјҡ

  • е®№еҷЁзұ»еһӢ
    е»әи®®дҪҝз”Ё chroot
    proot зӣ®еүҚеңЁlinuxdeployдёҚе®ҢеӨҮ,еҸҜиғҪж— жі•еңЁе®үиЈ…
  • еҸ‘иЎҢзүҲGNU/linux / еҸ‘иЎҢзүҲGNU/linux зүҲжң¬
    йҖүжӢ©дҪ йңҖиҰҒзҡ„зі»з»ҹпјҢжң¬жЎҲдҫӢйҖүжӢ©дәҶdebian пјҢзүҲжң¬stable
  • жһ¶жһ„
    йҖүжӢ©дҪ жүӢжңәж”ҜжҢҒзҡ„жһ¶жһ„пјҢеҸҜд»ҘзӮ№еҮ»еҸідёҠи§’дёүдёӘзӮ№пјҢйҖүжӢ©зҠ¶жҖҒпјҢиҫ“еҮәзҡ„дҝЎжҒҜдёӯ ArchitectureеҚідёәжң¬жңәcpuжһ¶жһ„пјҢдёҖиҲ¬й»ҳи®ӨеҚіеҸҜ
  • жәҗең°еқҖе»әи®®дҪҝз”ЁеӣҪеҶ…й•ңеғҸпјҢдёӢиҪҪеҝ«дёҖдәӣпјҢжң¬жЎҲдҫӢдҪҝз”Ёй•ңеғҸдёәпјҡhttp://mirrors.aliyun.com/debian/
  • е®үиЈ…зұ»еһӢ
  1. жң¬жЎҲдҫӢдҪҝз”ЁдәҶзӣ®еҪ•зұ»еһӢ
  2. й•ңеғҸж–Үд»¶
    е°ҶGNU/linuxе®үиЈ…еҲ°дёҖдёӘй•ңеғҸж–Үд»¶(иҷҡжӢҹзЈҒзӣҳ)дёӯ
    еҰӮжһңдҪ е°ҶGNU/linuxе®үиЈ…еҲ°fat32зҡ„sdеҚЎдёӯиҜ·дҪҝз”ЁиҜҘйҖүйЎ№
    е®үиЈ…и·Ҝеҫ„дёәй•ңеғҸж–Үд»¶зҡ„з»қеҜ№и·Ҝеҫ„ й»ҳи®Өдёә/sdcard/linux.img
    й•ңеғҸеӨ§е°Ҹдёәй•ңеғҸж–Үд»¶зҡ„еӨ§е°Ҹ й»ҳи®ӨдёәиҮӘеҠЁ
    еҰӮжһңй•ңеғҸи·Ҝеҫ„и®ҫзҪ®еңЁfat32еҶ…еӯҳеҚЎзҡ„зӣ®еҪ•дёӢй•ңеғҸеӨ§е°Ҹеә”дёҚеӨ§дәҺ4096mb (fat32ж–Үд»¶зі»з»ҹж–Үд»¶еӨ§е°ҸйҷҗеҲ¶)
    ж–Үд»¶зі»з»ҹдёәй•ңеғҸж–Үд»¶еҶ…иҰҒдҪҝз”Ёзҡ„еҲҶеҢәзҡ„ж–Үд»¶зі»з»ҹ й»ҳи®Өдёәext4
  3. зӣ®еҪ•
    е°ҶGNU/linuxе®үиЈ…еңЁдёҖдёӘext4жҲ–f2fsеҲҶеҢәзҡ„дёҖдёӘзӣ®еҪ•йҮҢ
    е®үиЈ…и·Ҝеҫ„дёәиҰҒе®үиЈ…еҲ°зҡ„зӣ®еҪ• й»ҳи®Өдёәlinuxdeployзҡ„еә”з”Ёж•°жҚ®зӣ®еҪ•(еңЁdataеҲҶеҢә)
    е®үиЈ…зӣ®еҪ•иҜ·еӢҝдҪҝз”Ёfat32еҲҶеҢәзҡ„зӣ®еҪ•,д»ҘеҸҠ/sdcard/ еҶ…йғЁе…ұдә«еӯҳеӮЁзӣ®еҪ•, е®ғд»¬дёҚж”ҜжҢҒlinuxж–Үд»¶зі»з»ҹзҡ„еҹәжң¬жқғйҷҗ
  4. еҲҶеҢә
    ж јејҸеҢ–дёҖдёӘеҲҶеҢәе№¶е°ҶGNU/Linuxе®үиЈ…еҲ°йҮҢйқў
    е®үиЈ…и·Ҝеҫ„дёәдёҖдёӘеқ—и®ҫеӨҮеҲҶеҢәж–Үд»¶зҡ„з»қеҜ№и·Ҝеҫ„ й»ҳи®Өдёә/dev/block/mmcblkXpY
    еҰӮжһңиҰҒе®үиЈ…еҲ°еҶ…еӯҳеҚЎзҡ„з¬¬дёҖдёӘеҲҶеҢәиҜ·еҶҷ: /dev/block/mmcblk1p1
    еҰӮжһңиҰҒе®үиЈ…еҲ°otg uзӣҳзҡ„з¬¬дёҖдёӘеҲҶеҢәиҜ·еҶҷ: /dev/block/sda1
    ж–Үд»¶зі»з»ҹж јејҸеҢ–еҲҶеҢәжүҖжҢҮе®ҡзҡ„ж–Үд»¶зі»з»ҹ й»ҳи®Өдёәext4
  5. RAM
    еҲӣе»әдёҖдёӘramdiskе№¶е°Ҷlinuxе®үиЈ…еҲ°йҮҢйқў
    е®үиЈ…и·Ҝеҫ„дёәramdiskзҡ„жҢӮиҪҪи·Ҝеҫ„ й»ҳи®Өдёә/data/local/ram
    й•ңеғҸеӨ§е°Ҹдёәramdiskзҡ„еӨ§е°Ҹ
    ramdiskдјҡеңЁе…іжңәеҗҺжё…йҷӨ
  • з”ЁжҲ·еҗҚ / з”ЁжҲ·еҜҶз Ғ
    з”ЁжҲ·еҗҚеЎ«еҶҷдёәrootпјҢз”ЁжҲ·еҜҶз ҒиҮӘе®ҡпјҢеҸӘжҳҜиҮӘе·ұзҺ©зҺ©зҡ„иҜқжҺЁиҚҗдҪҝз”Ё123456е°ұеҸҜд»ҘдәҶпјҢдёҚз„¶еҫҲе®№жҳ“еҝҳпјҢеҰӮжһңдҪ дёҚе°Ҹеҝғеҝҳи®°еҜҶз ҒпјҢеҸҜд»Ҙиҝ”еӣһжң¬ж•ҷзЁӢжҹҘзңӢжң¬ж•ҷзЁӢжҺЁиҚҗзҡ„еҜҶз ҒгҖӮ
  • зү№жқғз”ЁжҲ· root
  • DNS иҮӘеҠЁ
  • жң¬ең°еҢ–йҖүжӢ©пјҡzh_CN.UTF-8вҖҳ
