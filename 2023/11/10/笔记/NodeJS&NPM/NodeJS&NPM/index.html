npm常用命令

  • 项目初始化

    • npm init

      初始化当前文件夹（项目）的信息，生成package.json文件

    • npm init -y

      所有信息使用默认

  • 安装依赖

    • npm install 包名

      当前项目安装包

    • npm install 包名@版本号

      当前项目安装包带版本号

    • npm install

      根据package.json中的依赖信息安装依赖

    • npm install -g 包名

      全局库中安装包

    • npm ls

      查看当前项目依赖

    • npm ls

      查看当前项目依赖

    • npm run

      在package.json中的script属性
